1. Origin and Purpose
**Java**: - Developed by Sun Microsystems (now owned by Oracle). It was designed with the aim of creating a platform-independent, object-oriented programming language that could be used for a wide range of applications, including enterprise-level software, desktop applications, and Android app development (through the Android SDK which uses Java-based Android APIs). - It emphasizes strong typing, code maintainability, and performance. For example, many large-scale financial systems and business applications are built using Java due to its robustness and ability to handle complex business logic.
